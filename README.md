# Grok Automation — Releases

This repository hosts **public installer downloads** for Grok Automation.

Source code is private.

## Download

Latest release: [github.com/dongochoan97/grok_auto/releases/latest](https://github.com/dongochoan97/grok_auto/releases/latest)

### Windows
- `GrokAutomation-Setup-<version>-x64.exe` — Windows 10/11 x64

### macOS
- `GrokAutomation-<version>-arm64.dmg` — Apple Silicon (M1/M2/M3/M4)
- `GrokAutomation-<version>-x64.dmg` — Intel Mac

## First launch

### macOS
App is not notarized. On first launch:
1. Right-click the app in Applications → **Open**
2. Click **Open** in the dialog

### Windows
Windows SmartScreen may show a warning:
1. Click **More info**
2. Click **Run anyway**

## Auto-update

Once installed, the app checks for updates on startup. When a new version is
released, a notification appears in the bottom-left corner with a **Download**
button. Your choice whether to update.
